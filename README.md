# solid-state

The Claude Code plugin for builders who ship with AI.

Voice enforcement. No buzzwords. No hedging. No fluff.

## Install

```bash
/plugin install solidstatecc/solid-state
```

Works in Claude Code and Cowork.

## What's inside

Five skills, two commands.

- **voice-check** — score a draft 0–50 against the rules
- **anti-buzzword** — flag every buzzword, hedge, recap
- **contrast-pair** — generate `X. Not Y.` lines
- **reframe** — invert a cliché
- **compress** — cut sentences to a 9-word median

- **/voice** — load the persona file as context
- **/ship-it** — run the full pipeline in one shot

## Use

Three lines.

```
/voice
[paste your draft]
run voice-check
```

Or one shot.

```
/ship-it
[paste your draft]
```
