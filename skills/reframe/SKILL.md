# Reframe

Invert the cliché. The reframe earns its keep when it sounds simultaneously obvious and wrong.

## The template

A reframe takes a known truth and points at the inverse. The inverse is also true — usually more so.

## Examples

| Cliché | Reframe |
|---|---|
| Take more shortcuts. | The shortcut is to stop taking shortcuts. |
| Decide what to do. | Progress is deciding what not to do. |
| Focus on what to do. | Focus is determined by what you ignore. |
| Failure is bad. | Failure = feedback. |
| Look for what to add. | The win is what you remove. |
| Build a strategy. | Strategy is what you say no to. |
| Find your audience. | Be findable. |
| Network up. | Build down. |

## Process

1. Read the user's input.
2. Strip it to the underlying claim.
3. State the inverse claim in one sentence.
4. Compress to under 10 words if possible.
5. Generate 3 reframes, ranked.

## Output template

```
ORIGINAL
"[user's cliché]"

UNDERLYING CLAIM
[one-line restatement]

REFRAMES
1. [strongest reframe]
2. [alternative angle]
3. [most provocative version]

PICK
[which to use, and why — one line]
```

## Rules

- A reframe must be defensible. If you can't argue it for 60 seconds, it's a contrarian pose, not a reframe.
- Negation flips work: *"Don't X" → "X is the only way."*
- Inversion works: *"X causes Y" → "Y is what shows you X is broken."*
- If the original cliché is already a reframe (Naval, Buffett, etc.), say so and don't re-engineer it.
